# User Manual

## Using Groupdelegation as Sub Admin

If you are part of a sub-admin group, you have an additional icon next to your user settings.

![Sub-admin menu icon](../Images/subadmin_menu.png)

How to become a sub admin is explained in the [Administrator Manual](../Administrator/Index.md).

After clicking on "Groupdelegation", you will see a list of users whose permissions you can grant or withdraw. For
every user you can see the BE groups they belong to, divided into those you are allowed to remove and those you are not.

![Sub-admin user list](../Images/subadmin_userlist.png)

Click on the avatar or the button with the user name to edit a user.

At the top of the edit page, the user account can be disabled and start/stop times can be set.

Below is a list of all delegatable groups. They can be added by activating the checkbox next to each group.
Clicking **Save!** will add the selected groups to the user and remove the unselected ones.

![Sub-admin edit user](../Images/subadmin_edit-user.png)

Afterwards you will be redirected back to the editable users overview page.
