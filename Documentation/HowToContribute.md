# How to Contribute

## Contribution Rules

- There must be a ticket in the project's [bug tracker](https://github.com/in2code-de/groupdelegation/issues) explaining the problem or suggested enhancement
- [PSR-2](http://www.php-fig.org/psr/psr-2/) coding guidelines are enforced
- Commit messages must comply with the [format used by TYPO3](https://wiki.typo3.org/CommitMessage_Format_(Git)) (the "releases:" line is not required here)
- One unique logical change per patch
- The commit message must refer to the ticket number on GitHub, e.g. `Resolves #9`
